% How to add links for a new chip
% - Make a copy of an existing link block for a chip
% - Update chipname in \tagged
% - Update links, but DO NOT FORGET:
%     - Link to docs should start with https://espressif.com/documentation/...
%     - Escape all special chars in URLs, such as \_, \#, \?, etc.
% - Add % at the end of each link to avoid possible errors
% - Add new chipname to \definedchipnames in
%     - resources-[doc-type]__EN
%     - resources-[doc-type]__CN

% For complete instructions, go to
% https://www.overleaf.com/project/60e2ed1aefb70e595778b04b
% > Compile readme-general.tex
% > see "Add Section Related Documentation and Resources"
